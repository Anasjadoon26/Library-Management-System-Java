# Library Management System (Java)

This is a simple **Library Management System** developed in **Java** as a university project.  
The system supports both **Console-based** and **GUI-based (Java Swing)** interaction.

---

## Features

### Member Management
- Add new members
- Remove members
- Search member by ID
- Display all registered members (Console)

### Book Management
- Add books to inventory
- Remove books
- Search book by ID
- Display all books (Console)

### Issue / Return
- Issue books to members
- Return issued books
- Maximum issue limit per member is enforced

## How to Run
1. Open the project in **IntelliJ IDEA**
2. Run `Main.java`
3. GUI window will open
4. Console is used for detailed output and logs

## Notes
- This project is designed for learning **OOP concepts**
- No database is used (data is stored in memory)
- GUI and Console share the same data
